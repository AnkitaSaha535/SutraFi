🎯 About

SutraFi is a gamified, AI-powered financial planning platform built for Indian users. It simplifies complex financial decisions and makes wealth-building accessible to everyone.

95% of Indians lack structured financial planning — SutraFi bridges that gap with intelligent automation and engaging UX.

With a Money Heist–inspired experience, users navigate their financial journey like a strategic mission—clear goals, smart decisions, and calculated risks.
